# NEXT-ABSA (Aspect-Based Sentiment Analysis)


## Development Environment
This is for setting up your IDE development environment.
```sh
$ pipenv install --skip-lock -r requirements.txt 
```

To activate this project's virtualenv
```sh
$ pipenv shell
```
